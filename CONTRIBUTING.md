# Contributing to OFA Panel

Thank you for your interest in contributing! Please follow these guidelines to make collaboration smooth.

## Code Style
- Follow PSR-12 for PHP code.
- Use a linter/formatter (e.g., PHP CS Fixer) before submitting PRs.

## How to Contribute
1. Fork the repository
2. Create a topic branch for your change
3. Commit clear, focused changes with descriptive messages
4. Open a Pull Request against `main` and describe the change

## Reporting Issues
- Use GitHub Issues and provide steps to reproduce, expected behavior, and environment details.
